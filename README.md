#welcome to my bioargo project!
this repositiory contains modular code and data processing pipelines to download, clean and analyze biogeochemical argo float profiles. 
this is customized for three Indian Ocean Basins- Arabian Sea, Bay of Bengal and Equatorial IO. 
current active module is designed to automate data fetching for target regions; look for chlorophyll as a variable; clean it with QC filtering; limit to >20 cycles;interpolate on standard grid;create a depth resolved monhtly climatology and plot it with SST data fetched from NOAA OISST. 
growing this repository post code scrubbing to also include more methods of data fetching; analysis and plotting for CHL, NITRATE, DOXY, TEMP, SAL; comaprision of data with different QC filters; MLD, Nitricline and DCM analysis.
